# Commit格式
```txt=
[<Type>] <Title>
<Detail>
```
## Example
```txt=
[Perf] 逐行讀取效能提升
- FileStream增加buffer設定
- FileOptions選擇SequentialScan
```

## Type
|Type    |說明                   |
|--------|-----------------------|
|Feat    |新功能                 |
|Fix     |修 bug                 |
|Perf    |效能提升               |
|Refactor|重構、刪無用程式碼       |
|Style   |格式、風格（不含邏輯）   |
|Docs    |文件                   |
|Test    |測試                   |
|Build   |建置與依賴設定         |
|Ci      |CI/CD                  |
|Chore   |其他雜項               |
|Security|安全性修補             |
